YouTube 영상 처리 파이프라인과 검색 시스템의 모든 설정을 커스터마이징할 수 있습니다.

  • 🎤 자막 생성 설정: Gemini 모델 선택
  • 🌐 번역 설정: Gemini 모델 선택 및 번역 프롬프트 커스터마이징
  • 📝 요약 설정: 전체 영상 요약을 위한 프롬프트 설정
  • 🔍 벡터DB 필터링 설정: 중요 세그먼트 필터링 프롬프트
  • 🔎 검색 쿼리 확장 설정: 사용자 질문을 분석하여 검색 키워드 확장
  • 💬 검색 답변 생성 설정: 검색 결과를 바탕으로 최종 답변 생성
  • 🔧 일반 설정: Task 관리, 자동 클린업, HTML 생성 옵션 등

🎤 자막 생성 설정

자막 생성에 사용할 Whisper 모델을 선택하세요.

🌐 번역 설정

번역 및 요약에 사용할 Gemini 모델을 선택하세요. 기본값은 gemini-flash-latest (비용 효율적)입니다.
번역 AI의 역할과 전반적인 지시사항을 입력하세요.
{text}는 자막 텍스트로 자동 치환됩니다. 예: "다음 텍스트를 번역해줘: {text}"

📝 요약 설정

요약 AI의 역할과 전반적인 지시사항을 입력하세요.
{text}는 전체 자막 텍스트로 자동 치환됩니다.

🔍 벡터DB 필터링 설정

중요 세그먼트 필터링 AI의 역할과 지시사항을 입력하세요.
{text}는 세그먼트 목록으로 자동 치환됩니다. JSON 배열 형식으로 인덱스를 반환해야 합니다.

🔎 검색 쿼리 확장 설정

검색 쿼리 확장에 사용할 모델을 선택하세요. 기본값은 gemini-flash-latest (빠르고 저렴)입니다.
검색 쿼리를 확장하는 AI의 역할과 지시사항을 입력하세요. 사용자 질문을 분석하여 검색에 효과적인 키워드를 추가합니다.

💬 검색 답변 생성 설정

검색 결과를 바탕으로 최종 답변을 생성하는 모델을 선택하세요. 기본값은 gemini-flash-latest (빠르고 저렴)입니다.
검색 결과를 바탕으로 최종 답변을 생성하는 AI의 역할과 지시사항을 입력하세요. 답변의 톤, 길이, 형식 등을 지정합니다.

🔧 일반 설정

유지할 최대 Task 개수를 설정하세요. 0이면 제한 없음. 초과 시 선호도가 낮은 Task부터 자동 삭제됩니다. (요약보기 1점, 상세보기 3점, 추천 5점 기준)
비활성화하면 동일 URL의 중복 Task와 최대 개수 초과 시 자동 삭제가 실행되지 않습니다. 프롬프트 테스트 시 유용합니다.
상세보기 HTML 생성을 건너뛰면 처리 속도가 빨라집니다. 요약만 필요할 때 유용합니다.
검색 결과를 LLM으로 재평가하여 더 정확한 순위를 제공합니다. 비활성화하면 벡터 유사도만 사용하여 빠르지만 정확도가 낮을 수 있습니다.
검색 결과를 바탕으로 LLM이 요약 답변을 생성합니다. 비활성화하면 검색 결과만 유사도 순으로 표시하여 빠르고 비용이 절감됩니다.
Re-ranking 활성화 시 검색 결과를 필터링할 최소 유사도를 설정합니다. (0.0 ~ 1.0) 예: 0.5 = 50% 이상, 0.3 = 30% 이상. 낮을수록 더 많은 결과가 표시됩니다.

🔄 Qdrant 재인덱싱

완료된 모든 영상을 Qdrant DB에 재인덱싱합니다. 콘솔 로그를 확인하세요.

📄 HTML 파일 최신화

구버전 상세 HTML을 최신 형식으로 재생성합니다. 검색에서 정확한 위치로 이동하려면 필요합니다.
